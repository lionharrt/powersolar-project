🚀 QUICK START (60 seconds):

1. Extract to new project:
   $ cp -r dynamic-section-builder ~/my-new-project
   $ cd ~/my-new-project

2. Install dependencies:
   $ npm install

3. Run development server:
   $ npm run dev

4. Customize branding:
   - Edit src/locales/en/common.json

5. Deploy:
   $ npm run build
