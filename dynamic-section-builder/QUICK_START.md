# Quick Start - 60 Second Setup

## 1. Extract
```bash
cp -r dynamic-section-builder ~/my-new-project
cd ~/my-new-project
```

## 2. Install
```bash
npm install
```

## 3. Run
```bash
npm run dev
```

## 4. Customize
Edit `src/locales/en/common.json`:
```json
{
  "company": {
    "name": "Your Company Name",
    "email": "hello@yourcompany.com"
  }
}
```

## 5. Deploy
```bash
npm run build
```
